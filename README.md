# CKatas - C# Coding Katas

A collection of C# coding challenges (katas) for practicing programming fundamentals and problem-solving skills.

## Build & Run
```bash
dotnet build
dotnet test
dotnet run
```

## Technologies
- .NET 8
- xUnit (Testing Framework)
- C# 12

## Katas Included
1. **DescendingOrder** - Sort digits in descending order
2. **ToCamelCase** - Convert strings to camelCase format
3. **VowelCount** - Count vowels in a string
4. **FindNextSquare** - Find the next perfect square
5. **UniqueLetters** - Merge and sort unique letters from two strings
6. **CreatePhoneNumber** - Format array as phone number
7. **Pangram** - Detect if sentence contains all alphabet letters
8. **FindOutlier** - Find the odd one out in an array
9. **HighAndLow** - Find highest and lowest numbers in a string
